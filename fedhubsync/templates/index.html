{% extends "master.html" %} {% block title %} Home {% endblock %} {% block content %} How to use fedhubsync? ====================== To sync your repositoty on github to another host, once, you have deployed fedhubsync all you will need to add a git hook doing: ``http post http://127.0.0.1:5000/update/ key=123 repo=fedora-infra/bodhi -f`` For the fedora-infra/bodhi project. The ``key`` argument is an API key unique to the application and that ensures the project asking to be sync is allowed to do so. Server side that key is specified in the configuration file. {% endblock %}
